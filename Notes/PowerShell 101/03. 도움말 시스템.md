## ㄱ. 개요
- 전문가
    ```
    전문가들은 항상 답을 알지는 못하지만 답을 파악하는 방법을 알고 있습니다.
    PowerShell을 통해 도움말 시스템에 능숙해지는 것이 성공의 열쇠입니다.

    PowerShell 내에서 도움말 시스템을 사용하여 필요한 경우 명령을 검색하고 사용하는 방법을 배울 수 있습니다.
    PowerShell의 제작자인 제프리 스노버(Jeffrey Snover)가 여러 차례 비슷한 이야기를 나누는 것을 들었습니다.

    PowerShell에서 cmdlet은 "CMD-let" 아니라 "command-let" 으로 발음되는 컴파일된 명령입니다.
    cmdlet에 대한 명명 규칙은 쉽게 검색할 수 있도록 단수 동사-명사 형식을 따릅니다.
    예를 들어 Get-Process 실행 중인 프로세스를 결정하는 cmdlet이며, Get-Service 서비스 목록을 검색하는 cmdlet입니다.
    스크립트 cmdlet이라고도 하는 함수 및 별칭은 이 책의 뒷부분에서 설명하는 다른 유형의 PowerShell 명령입니다.
    "PowerShell 명령" 용어는 cmdlet, 함수 또는 별칭에 관계없이 PowerShell의 모든 명령을 설명합니다.

    ping.exe 및 ipconfig.exe같은 기존 명령줄 프로그램과 같은 PowerShell에서 운영 체제 네이티브 명령을 실행할 수도 있습니다.
    ```
* cmdlet: PowerShell 고유 명령

### 2장 - 도움말 시스템
#### 명령에 대한 정보를 검색할 수 있는 시스템
- 주요 명령
    - Get-Help
        - 명령의 상세 설명과 사용법
    - Help(=man, Get-Help)
        - Get-Help와 같으나 출력 화면을 조작
    - Update-Help
        - Get-Help 명령의 도움말을 인터넷으로 자동 설치
    - Get-Command
        - 명령 검색
    - Get-History
        - 현재 실행중인 파워셸의 입력 기록 출력
    - Start-Process
        - 프로세스를 실행
    - Get-EventLog
        - 특정 프로세스의 이벤트 로그 확인
    - more
        - 출력이 한 화면을 넘어가면, 스페이스나 화살표 키 등 화면 단위로 조작한다.

## ㄴ. 특징
- Get-Help 명령을 통해 사용 방법을 탐색 가능하다.
- Get-Command 명령을 통해 명령을 탐색 가능하다.

## ㄷ. 본문

### Get-Help
- 단일 매개변수(-Name)
    ```
    PS> Get-Help -Name Get-Help

    이름
    Get-Help

    개요
        Displays information about PowerShell commands and concepts.

    구문
        ...
    ```
    
- 매개변수 조작(-Full, -Detailed, -Examples, -Online, -Parameter Noun, -ShowWindow)
    ```
    PS> Get-Help -Name Get-Command -Full
    PS> Get-Help -Name Get-Command -Detailed
    PS> Get-Help -Name Get-Command -Examples
    PS> Get-Help -Name Get-Command -Online
    PS> Get-Help -Name Get-Command -Parameter Noun
    PS> Get-Help -Name Get-Command -ShowWindow
    ```

- 상세 설명(-Full)
    ```
    PS> Get-Help -Name Get-Help -Full

    ...
    매개 변수
    -Category [<System.String[]>]
        Displays help only for items in the specified category and their aliases. Conceptual articles
            are in
        the **HelpFile** category.
    ...
    ```
    - 출력 정보
        ```
        - 이름
        - 개요
        - 통사론
        - 설명
        - 매개 변수
        - 입력
        - 출력
        - 노트
        - 예제
        - 관련 링크
        ```

- 예제(-Examples)
    ```
    PS> Get-Help -Name History -Examples

    ...
    --------- Example 1: Get the session history ---------

        This example gets the entries in the session history. The default display shows each command and
        its
        ID, which indicates the order in which they ran.

        ```powershell
        Get-History
        ```
        ...
    ```

- 온라인 검색
    - -Online 옵션으로 해당 명령의 Microsoft Docs 웹 페이지를 열 수 있다.
        ```
        PS> Get-Help -Name History -Online
        ```

- 명령 찾기
    - 와일드카드 사용 위치에 따라 검색 범위가 다르다.
        ```
        PS> help *process*

        Name                              Category  Module                    Synopsis
        ----                              --------  ------                    --------
        Enter-PSHostProcess               Cmdlet    Microsoft.PowerShell.Core Connects to and enters into ...
        Exit-PSHostProcess                Cmdlet    Microsoft.PowerShell.Core Closes an interactive sessio...
        ...
        ```
    - "값 내에 와일드카드 문자를 지정하면 Get-Help 제공한 패턴과 일치하는 명령만 검색합니다."
        ```
        PS> help proces*
        ```
    - 값 외에 와일드카드 문자 사용시 검색이 가능하다.
        ```
        help *-process

        Name                              Category  Module                    Synopsis
        ----                              --------  ------                    --------
        Debug-Process                     Cmdlet    Microsoft.PowerShell.M... Debugs one or more processes...
        Get-Process                       Cmdlet    Microsoft.PowerShell.M... Gets the processes that are ...
        ...
        ```

### Update-Help
- 명령을 실행하면 인터넷을 통해 자동 설치한다.
    - 관리자 권한으로 실행해야 설치된다.
    ```
    PS> Update-Help
    ```

- 명령의 도움말을 새로운 창으로 제공한다.
    ```
    PS> Get-Help -Name history -ShowWindow
    ```

### Get-Command
- 명령 검색        
    * "도움말의 Get-Command 섹션에서는 이름, 명사및 동사 매개 변수가 와일드카드 문자를 허용한다는 것을 알 수 있습니다."
    - 속성 지정(-CommandType)
        ```
        PS> Get-Command -CommandType Alias

        CommandType     Name                                               Version    Source
        -----------     ----                                               -------    ------
        Alias           % -> ForEach-Object
        Alias           ? -> Where-Object
        Alias           ac -> Add-Content
        Alias           Add-AppPackage                                     2.0.1.0    Appx
        ...
        ```

    - 상세(-Full)
        ```
        help Get-Command -Full

        이름
            Get-Command

        개요
            Gets all commands.


        구문
        ...
        ```
    - 단일 지정 매개변수
        ```
        PS> Get-Command -Noun Process

        CommandType     Name                                               Version    Source
        -----------     ----                                               -------    ------
        Cmdlet          Debug-Process                                      3.1.0.0    Microsoft.PowerShell...
        ...
        ```
    - 다중 지정 매개변수
        ```
        PS> Get-Command -Name *service* -CommandType Cmdlet, Function, Alias, Script

        CommandType     Name                                               Version    Source
        -----------     ----                                               -------    ------
        Function        Get-NetFirewallServiceFilter                       2.0.0.0    NetSecurity
        Function        Set-NetFirewallServiceFilter                       2.0.0.0    NetSecurity
        Cmdlet          Get-Service                                        3.1.0.0    Microsoft.PowerShell...
        ...
        ```

- 이벤트 로그 명령어 사용 방법
    ```
    PS> Get-Help Get-EventLog

    Get-Help Get-EventLog

        Get-EventLog

    개요
        Gets the events in an event log, or a list of the event logs, on the local computer or remote
        computers.

    구문
        Get-EventLog [-After <System.DateTime>] [-AsBaseObject <System.Management.Automation.SwitchParame
        ter>] [-Before <System.DateTime>] [-ComputerName <System.String[]>] [-EntryType <System.String[]>
        ] [-Index <System.Int32[]>] [[-InstanceId] <System.Int64[]>] [-LogName] <System.String> [-Message
        <System.String>] [-Newest <System.Int32>] [-Source <System.String[]>] [-UserName <System.String[
        ]>] [<CommonParameters>]
    ...
    ```

### Help
- Get-Help와 같은 명령어
    - 스페이스 또는 Q 등으로 화면 조작이 가능하다.
    ```
    PS> Help Help

    이름
        Get-Help
    ...
    ```

### 도움말 없는 명령
- more
    - 일부 명령은 PowerShell 도움말 시스템에 등록되지 않았다.
    ```
    PS> help *more*
    ```
