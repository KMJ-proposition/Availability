## ㄴ. 본문
### PowerShell 원라이너
* CLI처럼 명령줄을 의미하는 PowerShell의 원라이너이다.
* 원라이너
    - 정상적인 원라이너
        ```PowerShell
        PS> Get-Service | Where-Object CanPauseAndContinue -eq $true | Select-Object -Property *


        Name                : ALUpdateService
        ...
        ...
        ```
        ```PowerShell
        PS> Get-Service | Select-Object -Property * | Where-Object CanPauseAndContinue -eq $true


        Name                : ALUpdateService
        ...
        ```
        ```PowerShell
        PS> Get-Service -Name W32Time | Select-Object -Property *           


        Name                : W32Time
        RequiredServices    : {}
        CanPauseAndContinue : False
        CanShutdown         : True
        CanStop             : True
        DisplayName         : Windows Time
        DependentServices   : {}
        MachineName         : .
        ServiceName         : W32Time
        ServicesDependedOn  : {}
        ServiceHandle       :
        Status              : Running
        ServiceType         : Win32OwnProcess, Win32ShareProcess
        StartType           : Manual
        Site                :
        Container           :
        ```
    - 원라이너가 아닌 경우
        ```PowerShell
        PS> $Service = 'w32time'; Get-Service -Name $Service

        Status   Name               DisplayName
        ------   ----               -----------
        Running  w32time            Windows Time
        ```
* 필터링
    - 비효율적인 경우
        ```PowerShell
        PS> Get-Service | Where-Object Name -eq w32time

        Status   Name               DisplayName
        ------   ----               -----------
        Running  W32Time            Windows Time
    ```
    - 효율적인 경우
        ```PowerShell
        PS> Get-Service -Name W32Time | Where-Object Name

        Status   Name               DisplayName
        ------   ----               -----------
        Running  W32Time            Windows Time
    ```
